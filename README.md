# Hand written digit recognizer form scratch

### Installation
```
install python3
```
```
# using pip3
pip3 install numpy 
```

## Quick start

- Clone the repo

- Run ```train.py``` to train the network
- Run ```test.py``` to test an image (copy your images to /images folder
